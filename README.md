# 🌌 Cloud T - Premium 互動式程式作品集與開發主控台

歡迎來到我的個人品牌作品集展示空間！這是一個專為全端開發者設計、採用極致暗色玻璃擬態（Glassmorphism）與螢光極光（Aurora）風格打造的互動式網頁產品。

🌐 **線上即時預覽 / Live Demo**：[https://asia17242.github.io/0529-L2-test/](https://asia17242.github.io/0529-L2-test/)

---

## 🎨 介面視覺亮點 (Visual & Interactive Features)

1. **HTML5 Canvas 粒子懸浮背景**：背景布滿數十顆高幀率（60fps）懸浮粒子，支援滑鼠游標引力與斥力的物理模擬，且會在鄰近粒子間自動勾勒精細的發光網格。
2. **極客風虛擬終端機 (Developer Console)**：內建完整的命令列模擬器，點選面板或輸入常用指令（如 `help`、`skills`、`projects`、`clear`）會觸發流暢的打字機效果日誌輸出。
3. **動態專案篩選藝廊 (Project Filter Grid)**：提供無刷新分類按鈕（前端設計、後端應用、實用工具、全部），專案卡片在切換時具備縮放與漸層淡入淡出動畫。
4. **磨砂玻璃詳情彈窗 (Details Modal)**：輕點任何專案卡片，即可展開精美的玻璃擬態彈窗，載入該專案的深度架構技術分析、技術標籤與連結。
5. **即時系統時鐘 (Digital Live Clock)**：以 `100ms` 高頻週期偵測系統時間，確保時、分、秒完美即時更新不延遲。
6. **模擬 GitHub 開發活動貢獻矩陣**：視覺化展示 GitHub Pixel 綠色格子牆，懸停時具有立體凸起放大與綠色發光呼吸動畫。
7. **全方位響應式網頁設計 (RWD)**：極佳的螢幕自適應排版，在手機、平板與寬螢幕桌機均有完美的卡片縮放與版面配置。

---

## 🛠️ 開發技術棧 (Tech Stack)

- **核心結構**：語意化 HTML5、動態 ES6+ JavaScript。
- **樣式與美學**：極致調校的 CSS3、高模糊值磨砂濾鏡 (`backdrop-filter`)、HSL 動態極光色彩系統。
- **第三方整合**：
  - **Google Fonts**：大氣標題字型 `Outfit`、質感內文字型 `Inter` 與程式碼等寬字型 `Fira Code`。
  - **Lucide Icons**：精美且高一致性的現代化向量圖示庫。
